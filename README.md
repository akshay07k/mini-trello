# Mini-Trello
A lightweight Trello clone built with [React, Node.js, Express, MongoDB, Socket.io].  
Features workspaces, boards, lists, cards, real-time updates, and authentication.


## Installation

### 1. Clone the repository
```bash
git clone https://github.com/akshay07k/mini-trello.git
cd mini-trello
```

### 2. Install dependencies
#### For Backend
```bash
cd server
npm install
npm run seed
```


#### For Frontend
```bash
cd ../client
npm install
```


### 3. Run application
#### Start backend:
```bash
cd server
npm run dev
```


#### Start frontend:
```bash
cd client
npm run dev
```
